# 个人简历 (Resume)

这是一个基于HTML和CSS的简洁个人简历模板，专为托管在GitHub Pages上设计。

## 如何使用

1. **编辑内容**: 打开 `index.html`，不仅可以修改文字内容，还可以根据注释替换你的个人信息、工作经历和技能。
2. **替换照片**: 
   - 将你的证件照或生活照重命名为 `profile_placeholder.png` 并覆盖当前目录下的同名文件。
   - 或者在 `index.html` 中修改 `<img src="...">` 的路径。
3. **调整样式**: 如果需要修改颜色或布局，可以编辑 `style.css`。

## 部署到 GitHub Pages

1. 将本项目推送到你的GitHub仓库。
2. 进入仓库的 **Settings** (设置) -> **Pages**。
3. 在 **Build and deployment** 下，选择 **Source** 为 `Deploy from a branch`。
4. 在 **Branch** 选项中，选择 `main` (或 `master`) 分支，文件夹选择 `/ (root)`。
5. 点击 **Save**。
6. 等待几分钟，GitHub会给出一个链接，你的在线简历就生成了！

## 主要特性

- 📱 **响应式设计**: 完美适配手机和电脑端。
- 🎨 **易于定制**: 清晰的代码结构和CSS变量。
- 🚀 **轻量级**: 纯HTML/CSS，无复杂框架依赖。
